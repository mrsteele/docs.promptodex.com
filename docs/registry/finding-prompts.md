# Finding Prompts

Discover prompts created by the community on the Promptodex registry.

## Browse on the website

Visit [promptodex.com](https://promptodex.com) to browse prompts by:

- **Category** — coding, writing, utility, business, fun, assistant, creative, and more
- **Popularity** — most used and most forked prompts
- **Search** — find prompts by keyword

## Use the search

The search bar at the top of [promptodex.com](https://promptodex.com) (also accessible with `⌘K`) lets you search across prompt titles, descriptions, and tags.

## Categories

Prompts are organized into categories:

| Category | Description |
|----------|-------------|
| `coding` | Code generation, review, and debugging |
| `writing` | Blog posts, articles, copy |
| `utility` | Summarization, translation, formatting |
| `business` | Emails, reports, analysis |
| `fun` | Creative and entertaining prompts |
| `assistant` | General-purpose helpers |
| `creative` | Art, storytelling, brainstorming |

## Bookmark prompts

Save interesting prompts to your bookmarks for quick access later. Click the bookmark icon on any prompt page.

## Fork prompts

Found a prompt that's close to what you need? Fork it, make your changes, and publish your own version.
