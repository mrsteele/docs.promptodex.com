# docs.promptodex.com

Documentation website for [Promptodex](https://promptodex.com) — Version Control for AI Prompts.

Built with [VitePress](https://vitepress.dev). All content is written in Markdown.

## Development

```bash
npm install
npm run docs:dev
```

## Build

```bash
npm run docs:build
npm run docs:preview
```

## Contributing

Every page has an **"Edit this page on GitHub"** link. Click it to suggest changes via pull request.
